Lenguaje de programación que he utilizado

HTML

Lenguaje de etiquetas de hipertexto

HTML no es un lenguaje de programación; es un lenguaje de marcado que define la estructura de tu contenido. Serie de elementos que usarás para encerrar distintas partes del contenido.

Saber más de HTML...

CSS

Hojas de Estilo en Cascada

Es un lenguaje informático que especifica cómo se presentan los documentos a los usuarios: cómo se diseñan, compaginan, etc. Permiten crear páginas web atractivas.

Saber más de CS...

JAVASCRIPT

JavaScript (abreviado comúnmente JS)

Es un lenguaje de secuencias de comandos que te permite crear contenido de actualización dinámica, controlar multimedia, animar imágenes y prácticamente todo lo demás.

Saber más de JS...

JAVA

La sigla JAVA es la sigla de Just Another Vague Acronym

Es un lenguaje de programación ampliamente utilizado para codificar aplicaciones web. Lenguaje multiplataforma, orientado a objetos y centrado en la red que se puede utilizar como una plataforma en sí mismo.

Saber más de JAVA...

PHP

Acrónimo recursivo de PHP: Hypertext Preprocessor

Es un lenguaje de código abierto muy popular especialmente adecuado para el desarrollo web y que puede ser incrustado en HTML.
